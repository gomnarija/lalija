##### c++ interpreter for lalija programming language.
## Installation

Clone the repo and make the project.
```sh
git clone https://github.com/gomnarija/lalija
cd lalija
make install
```

## Example
 ```
  laza>(function factorial (n) 
            (if (= n 1) n (* (factorial (-  n 1)) n)))
  factorial
  laza>(factorial 5)
  120
  laza>
```
